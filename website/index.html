About

I'm Sonder, an indie game developer passionate about creating unique, emotionally resonant experiences. I build games using Unity, focusing on tight mechanics, atmospheric design, and procedural generation.

When I'm not wrangling game objects, I'm diving into shader programming, tool development, and pushing the boundaries of what indie games can achieve. Every project is a chance to blend technical craft with artistic vision.

I believe great games come from iteration, attention to detail, and a deep understanding of what makes play feel good.
